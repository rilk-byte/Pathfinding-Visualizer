# Pathfinding-Visualizer
An interactive app built with Python and Tkinter which can represent how commonly used path-finding algorithms like BFS, DFS, Dijkstra, and A* work in real time

---

## Features

- Visualizes multiple algorithms:
  - Breadth-First Search (BFS)
  - Depth-First Search (DFS)
  - Dijkstra’s Algorithm
  - A* Search
  
- Interactive grid system
  - Set start and end points
  - Draw walls and weighted cells
    
- Real-time animation of algorithm traversal
- Adjustable speed
- Random maze generation

---

## Tech Stack

- Python
- Tkinter (GUI)
